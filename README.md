# go-keygen
Key generation package for golang

## Install

	go get github.com/mirrr/go-keygen

## Use

```golang
import (
	"fmt"
	"github.com/mirrr/go-keygen"
)
func main() {
	fmt.Println(keygen.NewKey(8))
}
```
